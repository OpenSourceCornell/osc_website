Come join OpenSourceCornell and ACSU to learn Git from the ground up! This workshop, based on [Tom Preston-Werner's Git Parable](http://tom.preston-werner.com/2009/05/19/the-git-parable.html), will help you build an understanding of how Git works and how to use it effectively. We won't be throwing lists of commands at you!

Topics covered:

- Commits
- Branching and tags
- Collaborating with others
- Working offline
- Merging
- Rewriting history
- Staging
- Diffs
- Working with Github
